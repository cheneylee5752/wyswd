### 诚挚欢迎史官们投稿！

投稿要求：
- 可以是新瓜也可以是老瓜的修正/补充。
- 以叙事视角行文，可以有立场但必须有理由据，提供必要截图/存档做证据，不能岁月史书。
- 格式不限，可以参考已有格式或者随意。


投稿方式：
1. 开一个issue，放稿件链接或直接写在issue里，热心网友会帮忙添加。
2. 直接提pull request。
